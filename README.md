## 📋 Descripción del Proyecto
Sistema de escritorio diseñado para administrar el inventario de negocios o almacenes de productos de limpieza. Permite registrar, modificar, eliminar y consultar productos, clasificados en 4 tipos: **Aerosoles, Líquidos, Sólidos y Utensilios de limpieza**.

El proyecto aplica los pilares fundamentales de la POO: **Encapsulamiento, Herencia, Polimorfismo y Abstracción**, además de integrar base de datos para guardar la información de forma permanente.
